# bearing analog rules

## Mandatory rules
- Exact bearing type match is required.
- Exact geometry match is required for a direct equivalent.
- If equivalence is not confirmed, return `NO DIRECT EQUIV`.
- Never map ball bearings to roller bearings or vice versa.

## Output discipline
- Separate confirmed facts from assumptions.
- State what is missing for confirmation.
- Show risk if a wrong analog is chosen.

## Safe inputs to ask for
- full designation
- d / D / B or T
- type / series
- seal / clearance / precision
- operating conditions

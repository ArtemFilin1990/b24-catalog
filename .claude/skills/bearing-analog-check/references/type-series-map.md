# bearing type and series map

## ISO families
- 60xx / 62xx / 63xx / 64xx / 16xxx -> deep groove ball
- 7xxx -> angular contact ball
- 22xxx / 23xxx -> spherical roller
- 30xxx / 31xxx / 32xxx (5-digit tapered families) -> tapered roller
- NU / NJ / N / NF / NUP -> cylindrical roller
- 51xxx-54xxx -> thrust ball
- 80xxx / 81xxx -> thrust roller

## Review reminder
Series hints are not enough when suffixes or local standard mappings change the interpretation.
Always validate against exact type and geometry before calling something a direct equivalent.
